# Student Grading System

A simple Student Grading System created using HTML, CSS, and JavaScript.

## Features
- Student Grading
- Engineering Cutoff Calculation
- CGPA Calculation
- Responsive black and gray theme

## How to Run
1. Download or save the project folder
2. Open the folder
3. Double click `index.html`
